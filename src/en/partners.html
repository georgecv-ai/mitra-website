Mitra, basic question. ∫ x·e^(x²) dx. I read the answer key but I don't get why a (1/2) shows up.
The x dx is hiding in the integral as a SET. Let x² = t, then 2x dx = dt, so x dx = dt/2. Replace it, only e^t is left.
