# How to Link to Mitra (with tracking)

*A short guide for partners and anyone sharing Mitra links. English first, Korean below. 한국어 안내는 아래에 있습니다.*

---

## English

### Why we ask you to use a special link

When you share Mitra with a normal link, we can't tell the visit came from you. When you use the tracking link below, we can see which of your posts brought families to Mitra, so we know what's working and can credit you properly. It takes 10 seconds per post.

### Your link

We give you a link that looks like this (yours will have your own name in it):

```
https://www.mitratutor.com/en/free-trial?utm_source=blog-supermom&utm_medium=referral&utm_campaign=partner-supermom&utm_content=____
```

**You only fill in one thing:** the blank at the end, `utm_content`. Put a short name for the specific post you're sharing it in. **Don't change anything else.**

### The one rule

Replace `____` with a short label for each post. For example:

| Where you're posting | Your link ends with |
|---|---|
| A blog review in March | `utm_content=march-review` |
| An Instagram reel | `utm_content=reel-adhd-tips` |
| Your newsletter | `utm_content=newsletter-may` |

So the finished link becomes:
```
https://www.mitratutor.com/en/free-trial?utm_source=blog-supermom&utm_medium=referral&utm_campaign=partner-supermom&utm_content=march-review
```

### How to name the post (3 small rules)

1. **All lowercase.** `reel-adhd-tips`, not `Reel ADHD Tips`.
2. **Use hyphens instead of spaces.** `back-to-school`, not `back to school`.
3. **Keep it short and descriptive.** No names, no emojis, no spaces.

### What NOT to change

- Don't change `utm_source`, `utm_medium`, or `utm_campaign`, those identify you.
- Don't shorten the link with a third-party shortener that strips the tags (if you must shorten, use a shortener that keeps the full URL).
- Don't remove the `?` or the `&` symbols.

### Korean visitors

If your audience is Korean, change `/en/` to `/ko/` in the link so they land on the Korean page:
```
https://www.mitratutor.com/ko/free-trial?utm_source=...&utm_content=...
```

### Questions

Email **support@mitratutor.com** and we'll help you build a link.

## 한국어

### 왜 전용 링크를 사용해야 하나요?

일반 링크로 Mitra를 공유하면, 그 방문이 어디에서 왔는지 저희가 알 수 없습니다. 아래의 추적 링크를 사용하시면, 어떤 게시물이 가정들을 Mitra로 데려왔는지 확인할 수 있어, 무엇이 효과가 있는지 파악하고 파트너님께 제대로 기여를 인정해 드릴 수 있습니다. 게시물당 10초면 됩니다.

### 파트너님의 링크

아래와 같은 형태의 링크를 드립니다 (실제로는 파트너님의 이름이 들어갑니다):

```
https://www.mitratutor.com/ko/free-trial?utm_source=blog-supermom&utm_medium=referral&utm_campaign=partner-supermom&utm_content=____
```

**채우실 곳은 한 군데뿐입니다:** 맨 끝의 빈칸 `utm_content`. 이 링크를 올리는 특정 게시물의 짧은 이름을 넣으세요. **나머지는 절대 변경하지 마세요.**

### 한 가지 규칙

게시물마다 `____` 자리에 짧은 이름을 넣으세요. 예시:

| 올리는 곳 | 링크 끝부분 |
|---|---|
| 3월 블로그 후기 | `utm_content=march-review` |
| 인스타그램 릴스 | `utm_content=reel-adhd-tips` |
| 뉴스레터 | `utm_content=newsletter-may` |

완성된 링크는 이렇게 됩니다:
```
https://www.mitratutor.com/ko/free-trial?utm_source=blog-supermom&utm_medium=referral&utm_campaign=partner-supermom&utm_content=march-review
```

### 게시물 이름 짓는 법 (작은 규칙 3가지)

1. **모두 소문자로.** `reel-adhd-tips` (O), `Reel ADHD Tips` (X)
2. **띄어쓰기 대신 하이픈(-).** `back-to-school` (O), `back to school` (X)
3. **짧고 알기 쉽게.** 이름·이모지·띄어쓰기는 넣지 마세요. 영문/숫자로 적어 주세요.

### 변경하면 안 되는 것

- `utm_source`, `utm_medium`, `utm_campaign`은 변경하지 마세요, 파트너님을 식별하는 정보입니다.
- 태그를 제거하는 단축 URL 서비스는 사용하지 마세요 (단축이 꼭 필요하면 전체 URL을 보존하는 서비스를 사용하세요).
- `?` 와 `&` 기호는 지우지 마세요.

### 영어권 방문자

대상이 영어권이라면 링크의 `/ko/` 를 `/en/` 로 바꿔 영어 페이지로 연결되도록 하세요:
```
https://www.mitratutor.com/en/free-trial?utm_source=...&utm_content=...
```

### 문의

**support@mitratutor.com** 으로 이메일 주시면 링크 만드는 것을 도와드리겠습니다.
