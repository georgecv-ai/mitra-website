# Logo assets

Canonical PNG files for the Mitra brand mark. All four are the current versions in use across the live site.

| File | Use |
|---|---|
| `mitra-icon.png` | The two-figure icon mark with the book. Use in nav, app surfaces, headers, anywhere the brand needs identification. |
| `mitra-wordmark.png` | "Mitra" wordmark in DM Sans 800, brand teal. Use in marketing display and partner co-branding. |
| `mitra-logo-transparent.png` | Generic transparent logo for layering over photographs or solid color. |
| `mitra-icon-128.png` | 128x128 square mark for favicons, app icons, social profile pictures. |

## Color variants

The files in this folder are the teal-on-transparent versions, designed to sit on white or light cream backgrounds. For use on dark backgrounds (the dark-teal nav, photography), request a white variant from the brand owner — we will produce one rather than letting partners recolor the logo themselves.

## Don'ts

- Do not recolor the logo to a non-brand color.
- Do not stretch, skew, rotate, or distort the proportions.
- Do not place the logo on busy backgrounds without enough clear space.
- Clear space rule: leave at least the height of the icon's circle as padding on all sides.
- Minimum size: 32px tall for digital, 12mm tall for print. Below those, the icon loses fidelity.

Vector versions exist in the source design files. Contact the brand owner for SVG / source files; PNGs in this pack are the publicly shareable copies.
